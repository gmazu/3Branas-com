Qué define hoy el marco

La prioridad actual no es “explicarlo todo” de una vez, sino cerrar piezas con mecanismo: geometría de colisión, escalas observacionales, perturbaciones del medio, analogías experimentales y documentos que puedan leerse sin depender de una conversación previa.

La colisión no se trata como una explosión sin mecanismo, sino como una interfaz física que barre una geometría.
eCEL se usa como régimen liberado del medio, con propagación, acoplamiento y respuesta ondulatoria.
La inflación aparente se lee geométricamente, no como materia viajando más rápido que la luz.
La misma arquitectura intenta conectar estructura cosmológica, ondas gravitacionales y ramas más microscópicas del modelo.

Estado actual

La prioridad sigue siendo cerrar documentos legibles, no inflar el sistema con nuevas ramas. La pregunta ya no es “¿hay una idea?”, sino qué parte está lo bastante cerrada como para leerse sin contexto extra.

Colisión de 2 3-Branas ya está en una fase de pulido y lectura externa.
MazzGAP funciona como una rama técnica separada para estabilidad y energía de enlace.
La web debe actuar como puerta de entrada, no como sustituto del paper.
